# 爬东财指数API

## 文档定位

**本文档详细说明如何从东方财富网获取指数的实时行情和历史数据**，包含完整的API参数、secid前缀说明、各指数URL和代码示例。

---

## 一、实时行情API

### 接口地址
```
https://{n}.push2.eastmoney.com/api/qt/stock/sse
```

> **说明**：`{n}` 为负载均衡数字（如56, 28, 9等），可任意使用

### 请求参数

| 参数 | 说明 | 示例值 | 必填 |
|------|------|--------|------|
| fields | 返回字段列表 | f58,f107,f57,f43,f59,f169,f170 | 是 |
| mpi | 每页数量 | 1000 | 是 |
| invt | 投资类型 | 2 | 是 |
| fltt | 复权类型 | 1 | 是 |
| secid | 证券ID（关键参数） | 124.HSCI, 100.HSI, 2.930914 | 是 |
| ut | 认证参数 | fa5fd1943c7b386f172d6893dbfba10b | 是 |

### 完整URL示例
```
https://56.push2.eastmoney.com/api/qt/stock/sse?fields=f58,f107,f57,f43,f59,f169,f170,f152,f46,f60,f44,f45,f171,f47,f86,f292&mpi=1000&invt=2&fltt=1&secid=124.HSCI&ut=fa5fd1943c7b386f172d6893dbfba10b&dect=1&wbp2u=7663325595358202|0|1|0|web
```

## 六、代码实现

### 实时行情获取
```python
import requests
import json

def fetch_eastmoney_realtime(secid):
    url = f"https://56.push2.eastmoney.com/api/qt/stock/sse"
    params = {
        'fields': 'f58,f107,f57,f43,f59,f169,f170,f152,f46,f60,f44,f45,f171,f47,f86,f292',
        'mpi': '1000',
        'invt': '2',
        'fltt': '1',
        'secid': secid,
        'ut': 'fa5fd1943c7b386f172d6893dbfba10b',
        'dect': '1',
        'wbp2u': '7663325595358202|0|1|0|web'
    }
    
    headers = {
        'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
        'Referer': 'https://quote.eastmoney.com/',
        'Accept': 'text/event-stream'
    }
    
    resp = requests.get(url, headers=headers, stream=True, timeout=10)
    for line in resp.iter_lines():
        if line:
            text = line.decode('utf-8')
            if text.startswith('data:'):
                json_str = text[5:]
                data = json.loads(json_str)
                if data.get('rc') == 0 and data.get('data'):
                    d = data['data']
                    return {
                        'name': d.get('f58'),
                        'symbol': d.get('f57'),
                        'last_price': d.get('f43', 0) / 100,
                        'prev_close': d.get('f60', 0) / 100,
                        'pct_change': d.get('f170', 0) / 100
                    }
    return None
```

### 历史数据获取
```python
def fetch_eastmoney_history(secid, days=30):
    url = 'https://push2his.eastmoney.com/api/qt/stock/kline/get'
    params = {
        'secid': secid,
        'fields1': 'f1,f2,f3,f4,f5,f6',
        'fields2': 'f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61',
        'klt': '101',
        'fqt': '1',
        'beg': '20260401',
        'end': '20261231',
        'ut': 'fa5fd1943c7b386f172d6893dbfba10b'
    }
    
    resp = requests.get(url, params=params, timeout=10)
    data = resp.json()
    
    if data.get('rc') == 0 and data.get('data'):
        klines = data['data'].get('klines', [])
        result = []
        for kline in klines:
            parts = kline.split(',')
            if len(parts) >= 6:
                result.append({
                    'date': parts[0],
                    'open': float(parts[1]),
                    'close': float(parts[2]),
                    'high': float(parts[3]),
                    'low': float(parts[4]),
                    'volume': int(parts[5])
                })
        return result
    return None
```

## 七、注意事项

1. **数据计算**：所有价格类字段都需要 ÷100 才能得到实际数值
2. **防刷机制**：建议每个指数请求间隔20秒，每日只同步一次
3. **URL域名**：push2.eastmoney.com 前面的数字是负载均衡标识，可任意使用
4. **返回格式**：实时行情接口返回SSE流格式，需解析 `data:` 前缀
