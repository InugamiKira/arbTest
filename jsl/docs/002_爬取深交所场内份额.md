# 爬取深交所场内份额

## 文档定位

**本文档详细说明如何从深圳证券交易所获取LOF基金的场内份额和净值数据**，包含完整的API参数、返回字段解析和代码示例。

---

## 一、API接口说明

### 接口地址
```
https://www.szse.cn/api/report/ShowReport/data
```

### 请求参数

| 参数 | 值 | 说明 | 必填 |
|------|-----|------|------|
| SHOWTYPE | JSON | 返回格式 | 是 |
| CATALOGID | 1945_LOF | LOF基金目录标识 | 是 |
| txtQueryKeyAndJC | 基金代码 | 如162411 | 是 |

### 完整URL示例
```
https://www.szse.cn/api/report/ShowReport/data?SHOWTYPE=JSON&CATALOGID=1945_LOF&txtQueryKeyAndJC=162411
```

---

## 二、返回格式

### 结构说明
```json
[
  {
    "metadata": {
      "catalogid": "1945_LOF",
      "name": "LOF基金列表",
      "subname": "2024-01-15"
    },
    "data": [
      {
        "sys_key": "<span>162411</span>",
        "kzjcurl": "<a href='...'>华宝油气</a>",
        "dqgm": "283,501.88",
        "zxjjjz": "0.9467"
      }
    ]
  }
]
```

### 字段解析

| 字段 | 说明 | 示例值 | 处理方式 |
|------|------|--------|----------|
| sys_key | 基金代码（含HTML标签） | `<span>162411</span>` | 去除HTML标签 |
| kzjcurl | 基金名称（含HTML标签） | `<a href='...'>华宝油气</a>` | 去除HTML标签 |
| dqgm | 当前规模（万份，含千分位） | `283,501.88` | 去除逗号后转浮点数 |
| zxjjjz | 最新基金净值 | `0.9467` | 直接转浮点数 |

## 三、代码实现

### Python示例
```python
import urllib.request
import urllib.parse
import json
import re

def fetch_szse_shares(fund_code, date):
    """
    从深交所获取LOF基金场内份额数据
    
    Args:
        fund_code: 基金代码（如162411）
        date: 查询日期（如'2024-01-15'）
    
    Returns:
        dict: {'code', 'name', 'shares', 'net_value'}
    """
    params = {
        'SHOWTYPE': 'JSON',
        'CATALOGID': '1945_LOF',
        'txtQueryKeyAndJC': fund_code
    }
    url = f"https://www.szse.cn/api/report/ShowReport/data?{urllib.parse.urlencode(params)}"
    
    req = urllib.request.Request(
        url, 
        headers={'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'}
    )
    
    try:
        with urllib.request.urlopen(req, timeout=10) as response:
            json_data = json.loads(response.read().decode('utf-8'))
            
            # 查找指定日期的数据
            for item in json_data:
                if item['metadata']['subname'] == date:
                    data = item['data'][0]
                    return {
                        'code': re.sub(r'<[^>]+>', '', data['sys_key']),
                        'name': re.sub(r'<[^>]+>', '', data['kzjcurl']),
                        'shares': float(data['dqgm'].replace(',', '')),  # 万份
                        'net_value': float(re.sub(r'<[^>]+>', '', data['zxjjjz']))
                    }
    except Exception as e:
        print(f"获取深交所数据失败: {e}")
    
    return None

# 使用示例
if __name__ == '__main__':
    result = fetch_szse_shares('162411', '2024-01-15')
    if result:
        print(f"基金代码: {result['code']}")
        print(f"基金名称: {result['name']}")
        print(f"场内份额: {result['shares']} 万份")
        print(f"最新净值: {result['net_value']}")
```

## 四、智能缓存机制（关键）

### 1. 时间限制
- **9:15前不请求**：深交所数据在9:15后才会更新，之前请求无意义
- **非交易日跳过**：周末和节假日数据不会更新

### 2. 请求频率控制
- **重复请求防护**：已有当日数据则跳过请求
- **缓存间隔**：20分钟（避免频繁请求被限流）
- **文件缓存**：建议缓存到 `debug/szse/{基金代码}.txt`

### 3. 完整缓存逻辑示例
```python
import os
import time

def should_fetch_szse(fund_code, date):
    """判断是否需要请求深交所数据"""
    
    # 时间检查：9:15前不请求
    now = time.localtime()
    current_minutes = now.tm_hour * 60 + now.tm_min
    if current_minutes < 9 * 60 + 15:  # 9:15前
        return False, "9:15前不请求"
    
    # 检查是否已有今日数据
    cache_dir = 'debug/szse'
    cache_file = os.path.join(cache_dir, f"{fund_code}.txt")
    
    if os.path.exists(cache_file):
        mtime = os.path.getmtime(cache_file)
        # 20分钟内的缓存有效
        if time.time() - mtime < 20 * 60:
            return False, "缓存未过期"
    
    return True, "需要请求"
```

### 4. 日期格式
- 查询日期格式必须为 `YYYY-MM-DD`
- 非交易日可能返回空数据或上一交易日数据

### 5. 数据单位
- `dqgm`（当前规模）单位为**万份**，无需再除以10000
- 示例：`283,501.88` 表示 283,501.88万份 = 28.35亿份

### 6. 历史数据获取
- 深交所API通常只返回最新日期的数据
- 获取历史份额数据需要提前每日保存

## 六、错误处理

### 常见错误

| 错误类型 | 原因 | 解决方法 |
|----------|------|----------|
| 空数据返回 | 日期格式错误或非交易日 | 检查日期格式，使用有效交易日 |
| 403 Forbidden | 请求被拒绝 | 添加User-Agent头部 |
| 网络超时 | 网络不稳定 | 添加超时重试机制 |

### 重试机制示例
```python
def fetch_szse_shares_with_retry(fund_code, date, max_retries=3):
    for retry in range(max_retries):
        result = fetch_szse_shares(fund_code, date)
        if result:
            return result
        print(f"第 {retry+1} 次重试...")
        time.sleep(2)
    return None
```
